호텔 예약 사이트 백엔드를 구축하려고 합니다. 아래 만족하는 API를 제작하세요.
 
회원정보: 아이디, 비밀번호, 이름, 성별, 가입날짜
호텔정보: 호텔코드, 호텔이름, 호텔위치, 전화번호, 웹사이트
호실정보: (호텔코드), 날짜, 시간, 가격, 호실설명
예약정보: 회원정보, 호텔정보, 호실정보, 예약날짜, 숙박날짜, 입실시간, 퇴실시간


회원가입 API

메소드: post
URL: /member/regist
반환: 성공, 중복아이디, 실패

* 데이터베이스는 RDBMS, NoSQL 중 선택
* node_modules는 삭제 후 업로드
* 모듈 설계는 자유롭게 작성
* 입력 매개변수에 대한 정보는 주석으로 자세히 작성
